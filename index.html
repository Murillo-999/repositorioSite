Passos para hospedar um projeto no github pages

1. crie um repositório no github com o mesmo nome do seu projeto
2. no terminal, inicialize o repositório local com git init
3. Altere o nome da branch principal para main
4. adicione os arquivos com git add . e faça o commit com git comit -m "first comit"
5. conecte ao repositorio remoto.com git remote add origin link
6. envie os arquivos com git init -m origin main
7. no github, via settings -> pages
8. em "Branch", selecione main
9. apos alguns segundos
